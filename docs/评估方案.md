# AgriWorld 模型评估方案说明

## 运行

```bash
cd /data4/Agri/yukaijie/AgriWorld/AgriWorld

# 默认评估 (使用 config.MODEL_VERSION 对应的 best checkpoint)
python evaluate.py

# 指定 checkpoint (如果 best 不是最新的)
python evaluate.py --ckpt saved_models/agriworld_lstm_res_final.pth

# 指定 GPU
python evaluate.py --device cuda
```

输出文件: `eval_baseline.pt` (序列化指标) + 终端打印报告。

## 评估指标判读标准

### 1) 产量精度 (bu/acre)

| 指标 | 公式 | 好 | 可接受 | 需改进 |
|------|------|:---:|:------:|:------:|
| **RMSE** | √(Σ(pred-tgt)²/n) | < 20 | 20–50 | > 50 |
| **NRMSE** | RMSE / mean(tgt) | < 10% | 10–25% | > 25% |
| **R²** | 1 - SS_res/SS_tot | > 0.7 | 0.3–0.7 | < 0.3 |
| **MAPE** | mean(|pred-tgt|/tgt) | < 10% | 10–20% | > 20% |
| **Bias** | mean(pred - tgt) | ±5 | ±5–15 | > ±15 |

**分层判读**: 分别看 Corn (crop_type=1), Soy (crop_type=5), Other。
- 如果 Corn bias 接近 0 但 Soy bias 很大 → 混合作物导致模型迷惑
- 如果所有组 bias 同号 → 系统偏差 (scale 或 HI 未校准)

### 2) 物理一致性 (validate_physics)

| 检查项 | 通过标准 | 含义 |
|--------|---------|------|
| **n_pool_range_valid** | 100% ∈ [0, 400] kg N/ha | N 池未超物理边界 |
| **water_balance_err** | < 0.5 | 闭合方程与 ODE 内部一致 |
| **lai_peak_valid** | 100% ∈ [120, 260] DOY | LAI 峰值应在生长季内 |
| **lai_max** | 3.0–8.0 | 玉米典型峰值 LAI |
| **sw_in_bounds** | 100% ∈ [-0.01, 0.61] | SW 未超 ODE 裁剪范围 |
| **bio_max** | 15–30 t/ha | 玉米典型最大生物量 |
| **n_pool_net_depletion** | 合理且有限 | 包含矿化、吸收和损失后的净变化 |

### 3) SMAP 探针

| R² | 判读 |
|:---:|------|
| > 0.5 | ODE 的 Soil_Water 编码了真实土壤水分信号 (PASS) |
| 0.2–0.5 | 有一定相关性但信号弱 (WARN) |
| < 0.2 | SW 是为优化产量产生的无意义数值 (FAIL) |

判据源自 `data_instrument.md` §5。

### 4) 参数收敛诊断

| 参数 | 初始值 | 期望终值 | 含义 | 如果未变化 |
|------|:------:|:--------:|------|-----------|
| **D0** | 2.000 | 1.0–3.0 | VPD 半衰减点 | 梯度未回传，学习率不够 |
| **gdd_flowering** | 850 | 700–1000 | 玉米开花积温 | 同上 |
| **gdd_maturity** | 1700 | 1400–2000 | 玉米成熟积温 | 同上 |
| **rue** | 3.0 | 2.5–4.0 | 辐射利用效率 | — |
| **k_ext** | 0.65 | 0.4–0.8 | 消光系数 | — |
| **sla** | 0.015 | 0.010–0.030 | 比叶面积 | — |
| **k_sen** | 0.008 | 0.005–0.020 | 衰老速率 | — |
| **log_vmax** | 0.41 | -0.7 ~ 1.1 | N 吸收最大速率(log kg N/ha/day) | — |
| **log_km** | 3.69 | 3.0 ~ 4.5 | M-M 半饱和常数(log kg N/ha) | — |
| **pedotransfer** 权重 | 随机 | — | 土壤质地→水力参数 | 由预训练初始化 |
| **coupling** 权重 | 零初始 | — | 胁迫融合 | 由预训练初始化 |

## 评估报告解读模板

运行 `python evaluate.py` 后，按以下顺序阅读输出：

```
Step 1: 看 产量精度 表
  → 总体 RMSE/R²/偏置
  → 玉米 vs 大豆 vs 其他 的分组差异

Step 2: 看 物理一致性
  → LAI 峰值在 DOY 多少？正常应在 DOY 180-250
  → 水分平衡误差多大？>2 说明 ODE 内方程不一致
  → SW 在边界内吗？

Step 3: 看 SMAP R²
  → >0.5 说明水文模块学对了

Step 4: 看 参数诊断
  → 关注 D0 / gdd_* 是否更新
  → 关注 log_vmax / log_km 是否合理
  → 关注 FC / WP 是否落于土壤质地文献范围

Step 5: 看 因子响应审计
  → Radiation、Nitrogen 不应显著为负
  → VPD 不应显著为正
  → Precipitation、Temperature 按环境背景解释
```
